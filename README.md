### 프로젝트 개요

한국관광공사에서 제공하는 TourAPI를 이용해 여행 계획에 필요한 관광지 정보를 제공하고,
원하는 관광지를 최단 시간으로 경유하는 루트를 제공해 여행 계획을 손쉽게 작성할 수 있도록 해주는
어플리케이션 입니다.

* [웹 페이지](http://52.79.131.13:8080) - 클라우드 컴퓨팅을 이용해 구축한 서버로, 과금으로 인해 stop시 접속이 제한될 수 있음.
* 현재 서버 상태 - `running`
